# Contributing

Use small branches:

```text
rns/encode-decode
rns/moduli-selection
cuda/one-modulus
cuda/multi-channel
integration/rns-linear
benchmark/latency
```

Before PR:

```bash
pytest
```

If blocked for more than 2 hours, send:

```text
BLOCKED:
What I tried:
What failed:
What I need:
```

Commit code every day. Keep PRs focused.
