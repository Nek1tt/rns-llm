# Experiment Results

This directory stores generated experiment outputs.

The source code does not include large benchmark artifacts by default.
After running the Colab/Kaggle workflow, place reproducible outputs here.

Structure:
- `v0.14.2/tables` - CSV/LaTeX tables for the paper
- `v0.14.2/plots` - figures
- `v0.14.2/benchmarks` - JSON/CSV benchmark results
- `v0.14.2/nsight` - Nsight Systems/Compute reports
